# Copilot Skills Library

A curated list of GitHub Copilot skills for full stack application development, workflows, and data. Create custom skills and place them under `.github/skills/` see how to create a custom skill [here](https://code.visualstudio.com/docs/agent-customization/agent-skills).

## Frontend

- **javascript-typescript-jest**  
  Best practices for JS/TS testing with Jest.  
  Install: `gh skill install github/awesome-copilot javascript-typescript-jest`  
  Documentation: [javascript-typescript-jest](https://awesome-copilot.github.com/skill/javascript-typescript-jest/)

- **frontend-testing**  
  Component testing, accessibility checks, and UI test structure for frontend apps.  
  Install: `gh skill install github/awesome-copilot frontend-testing`  
  Documentation: [frontend-testing](https://awesome-copilot.github.com/skill/frontend-testing/)

- **folder-structure-blueprint-generator**  
  Standardized frontend repository layouts and folder organization.  
  Install: `gh skill install github/awesome-copilot folder-structure-blueprint-generator`  
  Documentation: [folder-structure-blueprint-generator](https://awesome-copilot.github.com/skill/folder-structure-blueprint-generator/)

- **code-exemplars-blueprint-generator**  
  Capture reusable frontend code patterns and standards.  
  Install: `gh skill install github/awesome-copilot code-exemplars-blueprint-generator`  
  Documentation: [code-exemplars-blueprint-generator](https://awesome-copilot.github.com/skill/code-exemplars-blueprint-generator/)
