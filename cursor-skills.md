# Cursor Skills Library

A curated list of reusable Cursor Agent Skills for full-stack application development, workflows, and data. Store custom skills as `SKILL.md` folders and keep each skill focused on one domain-specific task.

## Repository Layout

```text

├── README.md
├── frontend/
├── backend/
├── devops/
├── data/
├── workflows/
├── architecture/
├── quality/
└── security/
```

## Skill Format

Each skill should live in its own folder and include a `SKILL.md` file.

```text
.cursor/skills/backend/python-fastapi-quality/
├── SKILL.md
├── examples/
└── references/
```

## What to Include in Each Skill

Each skill should usually contain:

- a short purpose statement,
- when to use it,
- what it should optimize for,
- rules and constraints,
- examples of good output,
- examples of bad output to avoid.

## Notes

Cursor describes Agent Skills as an open standard for packaging reusable knowledge and scripts. Keep each skill focused, concise, and domain-specific so the agent can load it only when relevant. [web:196][web:210]
